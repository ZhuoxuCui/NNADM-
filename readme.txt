% NNADMM++ : A MATLAB Toolbox for solving Nonconvex+Nonconvex structure
% variation problems for linear inverse problems with impulsive noiseI
% -------------------------------------------------------------------------
% Thank you for downloading the NNADMM++! 
% In this package the ADM is used to solve linear inverse problem 
% Ax = y^{\delta} direcly with a prior information that x is sparse  
% undertransformation L.  With a modified Morozov's discrepancy 
% principle as the stop condition in the paper, the ADM is a iterative 
% regularization method. And the included M files are used to reproduce 
% the figures in the paper.
